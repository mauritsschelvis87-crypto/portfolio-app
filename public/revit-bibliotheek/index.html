Revit Bibliotheek

Classificatiegestuurde desktop-app voor het doorzoeken, indienen en ordenen van Revit-families.

Windows desktop · WPF · netwerkschijf-bibliotheek

Wat doet het?

Revit Bibliotheek is een WPF-applicatie voor architectenbureaus die werken met een gedeelde Revit-bibliotheek op een netwerkschijf. De app bouwt een lokale SQLite-zoekindex zodat duizenden families snel doorzoekbaar zijn op NL-SfB-classificatie, categorie en metadata.

  • Zoeken en filteren op families in de bibliotheek
  • Handmatige sync met de netwerkschijf
  • Submit-wizard om nieuwe families toe te voegen
  • Revit-extractie-plugin voor metadata uit .rfa-bestanden
  • Materiaalbibliotheek-module (RevitMaterials)

Architectuur

Clean architecture met gescheiden lagen — UI, domain, infrastructure en Revit-bridge. Geen centrale server: de fileserver blijft bron van waarheid.

C# / .NET 10 WPF SQLite MVVM xUnit Docker CI

Waarom geen live demo?

Dit is een Windows-desktopapplicatie die gekoppeld is aan een interne netwerkschijf en Revit. Daarom toont deze pagina een projectoverzicht — de echte app draait lokaal bij gebruikers op hun werkplek.

De ontwikkelomgeving (Z:\-koppeling, appsettings, Revit-add-in) staat los van deze portfolio-pagina.
